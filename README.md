# MetaLinkKiller

## About

This program was created to kill the existing OVR processes after shutting down the Meta Horizon app, instead of having to kill them manually in the Task Manager.

Once opened it only exists in the System Tray

## Contents in the System Tray

- Kill OVR Process
- About
- Exit
- Run at Startup (Checkbox, simply starts the program)
- Kill on Startup (Checkbox)
